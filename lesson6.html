Здесь мы научимся использовать циклы - конструкции, которые позволяют выполнять код многократно. Это критически важный навык для автоматизации повторяющихся задач и обработки больших объемов данных.

Наша задача - освоить различные типы циклов и научиться эффективно применять их для решения практических задач.

Что такое циклы?

Циклы - это конструкции, которые позволяют выполнять блок кода многократно. Они помогают автоматизировать повторяющиеся задачи и значительно сокращают объем кода.

Аналогия: Представьте, что вам нужно пересчитать 1000 звезд. Вместо того, чтобы писать 1000 строк кода для подсчета каждой звезды, вы можете использовать цикл, который автоматически повторит процесс подсчета для всех звезд.

Цикл while

Цикл while выполняет блок кода, пока указанное условие истинно:

# Простой пример цикла while
count = 1

while count <= 5:
    print(f"Счетчик: {count}")
    count += 1 # Увеличиваем счетчик

print("Цикл завершен!")
Терминал
Счетчик: 1
Счетчик: 2
Счетчик: 3
Счетчик: 4
Счетчик: 5
Цикл завершен!

Цикл while проверяет условие перед каждой итерацией. Если условие истинно, выполняется блок кода. Если условие ложно, цикл завершается.

Внимание: Важно обеспечить, чтобы условие в какой-то момент стало ложным, иначе цикл будет выполняться бесконечно (бесконечный цикл). В примере выше мы увеличиваем count на каждой итерации, чтобы в конечном итоге условие count <= 5 стало ложным.

Пример: Ввод пароля

# Проверка пароля с использованием цикла while
correct_password = "космос123"
password = ""
attempts = 0
max_attempts = 3

while password != correct_password and attempts < max_attempts:
    password = input("Введите пароль: ")
    attempts += 1
    
    if password != correct_password and attempts < max_attempts:
        print(f"Неверный пароль! Осталось попыток: {max_attempts - attempts}")

if password == correct_password:
    print("Доступ разрешен. Добро пожаловать!")
else:
    print("Превышено количество попыток. Доступ заблокирован.")

Введите пароль: космос
Неверный пароль! Осталось попыток: 1
Введите пароль: космос123

Цикл for

Цикл for используется для итерации по последовательности (списку, строке, диапазону и т.д.):

# Простой пример цикла for с range()
for i in range(5): # range(5) создает последовательность [0, 1, 2, 3, 4]
    print(f"Итерация {i}")

print("Цикл завершен!")

Итерация 1
Итерация 2
Итерация 3
Итерация 4
Цикл завершен!

Функция range()

Функция range() создает последовательность чисел и часто используется с циклом for:

# range(stop) - от 0 до stop-1
for i in range(3):
    print(i) # Выведет: 0, 1, 2

# range(start, stop) - от start до stop-1
for i in range(2, 5):
    print(i) # Выведет: 2, 3, 4

# range(start, stop, step) - от start до stop-1 с шагом step
for i in range(0, 10, 2):
    print(i) # Выведет: 0, 2, 4, 6, 8

Итерация по последовательностям

Цикл for может итерировать по различным типам последовательностей:

# Итерация по строке
for char in "Привет":
    print(char)

# Итерация по списку
planets = ["Меркурий", "Венера", "Земля", "Марс"]
for planet in planets:
    print(f"Планета: {planet}")

Функция enumerate()

Если вам нужен индекс элемента во время итерации, используйте функцию enumerate():

planets = ["Меркурий", "Венера", "Земля", "Марс"]

for index, planet in enumerate(planets):
    print(f"Планета #{index + 1}: {planet}")

Управление циклами: break и continue

Оператор break

Оператор break используется для досрочного выхода из цикла:

# Поиск планеты в списке
planets = ["Меркурий", "Венера", "Земля", "Марс", "Юпитер"]
target = "Земля"

for planet in planets:
    print(f"Проверяем планету: {planet}")
    if planet == target:
        print(f"Планета {target} найдена!")
        break # Выходим из цикла, как только нашли нужную планету
Терминал

Проверяем планету: Земля
Планета Земля найдена!

Оператор continue

Оператор continue используется для перехода к следующей итерации цикла, пропуская оставшийся код в текущей итерации:

# Вывод только четных чисел
for i in range(1, 11):
    if i % 2 != 0: # Если число нечетное
        continue # Пропускаем оставшийся код и переходим к следующей итерации
    print(f"Четное число: {i}")

Цикл while vs. цикл for

Цикл while Цикл for
Выполняется, пока условие истинно Выполняется для каждого элемента в последовательности
Подходит, когда количество итераций заранее неизвестно Подходит, когда количество итераций известно или нужно перебрать все элементы
Требует ручного изменения переменной для завершения цикла Автоматически переходит к следующему элементу
Может привести к бесконечному циклу, если условие никогда не станет ложным Завершается после перебора всех элементов

Вложенные циклы

Циклы могут быть вложены друг в друга для решения более сложных задач:

# Таблица умножения с помощью вложенных циклов
for i in range(1, 6): # Внешний цикл: от 1 до 5
    for j in range(1, 6): # Внутренний цикл: от 1 до 5
        print(f"{i} × {j} = {i * j}", end="\t")
    print() # Переход на новую строку после завершения внутреннего цикла

Внимание: Вложенные циклы могут быть вычислительно затратными, особенно при большом количестве итераций. Используйте их с осторожностью и старайтесь оптимизировать код, если это возможно.

sum_result = 0

for i in range(1, n + 1):
    sum_result += i

print(f"Сумма чисел от 1 до {n} равна {sum_result}")

Пример 3: Поиск простых чисел

# Поиск простых чисел в заданном диапазоне
start = int(input("Введите начало диапазона: "))
end = int(input("Введите конец диапазона: "))

print(f"Простые числа в диапазоне от {start} до {end}:")

for num in range(start, end + 1):
    # Пропускаем числа меньше 2, так как они не являются простыми
    if num < 2:
        continue
    
    # Проверяем, делится ли число на какое-либо число от 2 до sqrt(num)
    is_prime = True
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            is_prime = False
            break
    
    if is_prime:
        print(num, end=" ")

🚀 Задание для завершения миссии

Для получения кода доступа к следующей планете, создай программу "Космический счетчик", которая:

  1. Запрашивает у пользователя количество планет для обратного отсчета
  2. Запрашивает у пользователя шаг обратного отсчета (сколько планет пропускать)
  3. Выводит обратный отсчет, пропуская указанное количество планет
  4. После достижения нуля выводит сообщение о запуске

Дополнительные требования:

  • Если пользователь вводит отрицательное число планет или шаг, программа должна вывести сообщение об ошибке и запросить данные снова
  • Если шаг равен 0, программа должна вывести сообщение об ошибке и запросить данные снова
  • Программа должна использовать цикл while для проверки корректности ввода и цикл for для обратного отсчета
