Здесь мы научимся создавать и использовать функции - блоки кода, которые можно вызывать многократно. Функции позволяют структурировать программу, избегать повторения кода и делать его более читаемым и поддерживаемым.

Наша задача - освоить создание функций и научиться эффективно применять их для решения практических задач.

Что такое функции?

Функции - это блоки кода, которые выполняют определенную задачу и могут быть вызваны многократно. Они позволяют разбить программу на логические части, что делает код более организованным и легким для понимания.

Аналогия: Представьте, что вы строите космический корабль. Вместо того, чтобы каждый раз создавать двигатель с нуля, вы разрабатываете его один раз и затем используете эту "функцию" в разных кораблях. Функции в программировании работают так же - вы пишете код один раз и используете его многократно.

calculate_area(radius)
Параметр: radius
Вычисляет площадь круга
Возвращает: площадь
Вызов функции: calculate_area(5)
Результат: 78.54

Определение функций

В Python функция определяется с помощью ключевого слова def, за которым следует имя функции и круглые скобки. Параметры функции указываются в скобках:

# Определение простой функции без параметров
def greet():
    print("Привет, космический путешественник!")

# Вызов функции
greet()
Терминал
Привет, космический путешественник!

Функции с параметрами

Функции могут принимать параметры - значения, которые передаются в функцию при ее вызове:

# Функция с одним параметром
def greet_by_name(name):
    print(f"Привет, {name}!")

# Вызов функции с аргументом
greet_by_name("Алекс")
greet_by_name("Капитан")
Терминал
Привет, Алекс!
Привет, Капитан!

Терминология: Параметры - это переменные, указанные в определении функции. Аргументы - это значения, которые передаются функции при ее вызове.

Функции с несколькими параметрами

# Функция с несколькими параметрами
def calculate_fuel(distance, consumption):
    fuel_needed = distance * consumption
    print(f"Для путешествия на {distance} световых лет потребуется {fuel_needed} единиц топлива.")

# Вызов функции с двумя аргументами
calculate_fuel(100, 0.5)
Терминал
Для путешествия на 100 световых лет потребуется 50.0 единиц топлива.

Возвращение значений из функций

Функции могут возвращать значения с помощью ключевого слова return. Это позволяет использовать результат работы функции в других частях программы:

# Функция, возвращающая значение
def calculate_area(radius):
    pi = 3.14159
    area = pi * radius ** 2
    return area

# Использование возвращаемого значения
circle_area = calculate_area(5)
print(f"Площадь круга с радиусом 5 равна {circle_area:.2f}")
Терминал
Площадь круга с радиусом 5 равна 78.54

Функция может возвращать несколько значений, которые будут упакованы в кортеж:

# Функция, возвращающая несколько значений
def get_coordinates():
    x = 10
    y = 20
    z = 30
    return x, y, z

# Получение нескольких значений
x, y, z = get_coordinates()
print(f"Координаты: ({x}, {y}, {z})")
Терминал
Координаты: (10, 20, 30)

Параметры по умолчанию

Функции могут иметь параметры с значениями по умолчанию. Если при вызове функции не указан аргумент для такого параметра, будет использовано значение по умолчанию:

# Функция с параметром по умолчанию
def greet_astronaut(name, rank="Космонавт"):
    print(f"Приветствую, {rank} {name}!")

# Вызов с обоими аргументами
greet_astronaut("Иванов", "Капитан")

# Вызов только с обязательным аргументом
greet_astronaut("Петров")
Терминал
Приветствую, Капитан Иванов!
Приветствую, Космонавт Петров!

Важно: Параметры с значениями по умолчанию должны идти после параметров без значений по умолчанию. Иначе Python выдаст ошибку синтаксиса.

Именованные аргументы

При вызове функции можно явно указывать имена параметров. Это делает код более читаемым и позволяет передавать аргументы в любом порядке:

def calculate_trip(distance, speed, fuel_consumption):
    time = distance / speed
    fuel = distance * fuel_consumption
    print(f"Время в пути: {time:.2f} часов")
    print(f"Требуется топлива: {fuel:.2f} единиц")

# Вызов с позиционными аргументами
calculate_trip(100, 20, 0.5)

# Вызов с именованными аргументами
calculate_trip(speed=25, distance=150, fuel_consumption=0.4)
Терминал
Время в пути: 5.00 часов
Требуется топлива: 50.00 единиц
Время в пути: 6.00 часов
Требуется топлива: 60.00 единиц

Переменное количество аргументов

Иногда нужно создать функцию, которая принимает произвольное количество аргументов. В Python это можно сделать с помощью специальных синтаксических конструкций:

*args - произвольное количество позиционных аргументов

def calculate_average(*args):
    total = sum(args)
    count = len(args)
    return total / count if count > 0 else 0

# Вызов с разным количеством аргументов
print(f"Среднее из 1, 2, 3: {calculate_average(1, 2, 3)}")
print(f"Среднее из 10, 20: {calculate_average(10, 20)}")
print(f"Среднее из 5, 10, 15, 20, 25: {calculate_average(5, 10, 15, 20, 25)}")
Терминал
Среднее из 1, 2, 3: 2.0
Среднее из 10, 20: 15.0
Среднее из 5, 10, 15, 20, 25: 15.0

**kwargs - произвольное количество именованных аргументов

def create_spaceship(**kwargs):
    print("Создание космического корабля со следующими параметрами:")
    for key, value in kwargs.items():
        print(f"- {key}: {value}")

# Вызов с разными именованными аргументами
create_spaceship(name="Звездный скиталец", engine="Ионный", speed="0.5c")

create_spaceship(name="Галактический странник", crew=5, weapons="Фотонные торпеды", shields="Активные")

Область видимости переменных

Переменные, определенные внутри функции, имеют локальную область видимости и не доступны за пределами функции:

def calculate_speed():
    distance = 100 # Локальная переменная
    time = 5 # Локальная переменная
    speed = distance / time
    print(f"Скорость: {speed} км/ч")

calculate_speed()

# Попытка доступа к локальной переменной вызовет ошибку
# print(distance) # NameError: name 'distance' is not defined
Терминал
Скорость: 20.0 км/ч

Глобальные переменные (определенные вне функций) доступны внутри функций, но для их изменения нужно использовать ключевое слово global:

total_distance = 0 # Глобальная переменная

def add_distance(distance):
    global total_distance # Указываем, что хотим изменить глобальную переменную
    total_distance += distance
    print(f"Добавлено {distance} км. Общее расстояние: {total_distance} км")

add_distance(100)
add_distance(50)
add_distance(75)

print(f"Итоговое расстояние: {total_distance} км")
Терминал
Добавлено 100 км. Общее расстояние: 100 км
Добавлено 50 км. Общее расстояние: 150 км
Добавлено 75 км. Общее расстояние: 225 км
Итоговое расстояние: 225 км

Совет: Старайтесь избегать использования глобальных переменных и ключевого слова global. Вместо этого лучше передавать значения через параметры и возвращать результаты через return. Это делает код более модульным и легким для тестирования.

Рекурсивные функции

Рекурсивная функция - это функция, которая вызывает сама себя. Рекурсия полезна для решения задач, которые можно разбить на подзадачи того же типа:

# Рекурсивная функция для вычисления факториала
def factorial(n):
    if n <= 1:
        return 1
    else:
        return n * factorial(n - 1)

# Вычисление факториала
print(f"5! = {factorial(5)}")
print(f"3! = {factorial(3)}")
Терминал
5! = 120
3! = 6

Внимание: При использовании рекурсии важно определить базовый случай (условие выхода из рекурсии), иначе функция будет вызывать себя бесконечно, что приведет к ошибке переполнения стека.

Лямбда-функции

Лямбда-функции (или анонимные функции) - это небольшие функции, которые можно определить в одной строке. Они полезны, когда нужна простая функция, которая используется только один раз:

# Обычная функция
def square(x):
    return x ** 2

# Эквивалентная лямбда-функция
square_lambda = lambda x: x ** 2

print(f"square(5) = {square(5)}")
print(f"square_lambda(5) = {square_lambda(5)}")
Терминал
square(5) = 25
square_lambda(5) = 25

Лямбда-функции часто используются с функциями высшего порядка, такими как map(), filter() и sorted():

# Использование лямбда-функции с map()
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))
print(f"Квадраты чисел: {squared}")

# Использование лямбда-функции с filter()
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(f"Четные числа: {even_numbers}")

# Использование лямбда-функции с sorted()
planets = [
    {"name": "Земля", "radius": 6371},
    {"name": "Марс", "radius": 3389},
    {"name": "Юпитер", "radius": 69911},
    {"name": "Меркурий", "radius": 2439}
]

sorted_planets = sorted(planets, key=lambda planet: planet["radius"])
for planet in sorted_planets:
    print(f"{planet['name']}: {planet['radius']} км")

Практические примеры

Пример 1: Конвертер температуры

def celsius_to_fahrenheit(celsius):
    return (celsius * 9/5) + 32

def fahrenheit_to_celsius(fahrenheit):
    return (fahrenheit - 32) * 5/9

def kelvin_to_celsius(kelvin):
    return kelvin - 273.15

def celsius_to_kelvin(celsius):
    return celsius + 273.15

# Функция-конвертер
def convert_temperature(value, from_unit, to_unit):
    # Сначала конвертируем в Цельсии
    if from_unit.upper() == "C":
        celsius = value
    elif from_unit.upper() == "F":
        celsius = fahrenheit_to_celsius(value)
    elif from_unit.upper() == "K":
        celsius = kelvin_to_celsius(value)
    else:
        return "Неизвестная единица измерения"
    
    # Затем конвертируем из Цельсия в нужную единицу
    if to_unit.upper() == "C":
        return celsius
    elif to_unit.upper() == "F":
        return celsius_to_fahrenheit(celsius)
    elif to_unit.upper() == "K":
        return celsius_to_kelvin(celsius)
    else:
        return "Неизвестная единица измерения"

# Примеры использования
print(f"0°C = {convert_temperature(0, 'C', 'F'):.2f}°F")
print(f"98.6°F = {convert_temperature(98.6, 'F', 'C'):.2f}°C")
print(f"300K = {convert_temperature(300, 'K', 'C'):.2f}°C")
print(f"25°C = {convert_temperature(25, 'C', 'K'):.2f}K")
Терминал
0°C = 32.00°F
98.6°F = 37.00°C
300K = 26.85°C
25°C = 298.15K

Пример 2: Калькулятор расстояния между планетами

import math

def calculate_distance(x1, y1, z1, x2, y2, z2):
    # Вычисляем расстояние между двумя точками в трехмерном пространстве
    return math.sqrt((x2 - x1)**2 + (y2 - y1)**2 + (z2 - z1)**2)

def get_planet_coordinates(planet_name):
    # Словарь с координатами планет (условные значения)
    planets = {
        "меркурий": (0.39, 0, 0),
        "венера": (0.72, 0, 0),
        "земля": (1.0, 0, 0),
        "марс": (1.52, 0, 0),
        "юпитер": (5.2, 0, 0),
        "сатурн": (9.54, 0, 0),
        "уран": (19.2, 0, 0),
        "нептун": (30.06, 0, 0)
    }
    
    return planets.get(planet_name.lower(), None)

def calculate_travel_time(distance, speed):
    # Вычисляем время путешествия в днях
    # Расстояние в а.е., скорость в а.е./день
    return distance / speed

def planet_distance_calculator():
    print("Калькулятор расстояния между планетами")
    print("Доступные планеты: Меркурий, Венера, Земля, Марс, Юпитер, Сатурн, Уран, Нептун")
    
    planet1 = input("Введите название первой планеты: ")
    planet2 = input("Введите название второй планеты: ")
    
    coords1 = get_planet_coordinates(planet1)
    coords2 = get_planet_coordinates(planet2)
    
    if coords1 is None or coords2 is None:
        print("Одна из планет не найдена. Проверьте правильность ввода.")
        return
    
    x1, y1, z1 = coords1
    x2, y2, z2 = coords2
    
    distance = calculate_distance(x1, y1, z1, x2, y2, z2)
    
    print(f"\nРасстояние между планетами {planet1.capitalize()} и {planet2.capitalize()}: {distance:.2f} а.е.")
    print(f"Это примерно {distance * 149.6:.2f} миллионов километров.")
    
    # Расчет времени путешествия для разных скоростей
    speeds = {
        "Космический зонд (0.001 а.е./день)": 0.001,
        "Космический корабль (0.005 а.е./день)": 0.005,
        "Быстрый корабль (0.01 а.е./день)": 0.01
    }
    
    print("\nРасчетное время путешествия:")
    for name, speed in speeds.items():
        days = calculate_travel_time(distance, speed)
        print(f"- {name}: {days:.1f} дней ({days/365:.2f} лет)")

# Запуск калькулятора
planet_distance_calculator()

Для получения кода доступа к следующей планете, создай программу "Космический калькулятор", которая будет содержать набор полезных функций для космических расчетов. Программа должна:

  1. Содержать функцию для расчета силы гравитации между двумя объектами (F = G * (m1 * m2) / r²), где G - гравитационная постоянная (6.67430e-11), m1 и m2 - массы объектов, r - расстояние между ними
  2. Содержать функцию для расчета времени падения объекта на поверхность планеты с заданной высоты (t = √(2h/g), где h - высота, g - ускорение свободного падения)
  3. Содержать функцию для расчета скорости, необходимой для выхода на орбиту планеты (v = √(G * M / r), где G - гравитационная постоянная, M - масса планеты, r - радиус орбиты от центра планеты)
  4. Содержать функцию для перевода между различными единицами измерения (километры в световые годы, метры в секунду в километры в час и т.д.)
  5. Иметь главное меню, которое позволяет пользователю выбрать нужный расчет
