Зачем нужны файлы?

Файлы позволяют программам сохранять данные на долгосрочной основе. В отличие от переменных, которые существуют только во время выполнения программы, файлы сохраняют информацию даже после завершения работы программы или выключения компьютера.

Основные причины использования файлов:

  • Сохранение состояния программы между запусками
  • Обработка больших объемов данных, которые не помещаются в оперативную память
  • Обмен данными между разными программами
  • Хранение конфигурационных настроек
  • Логирование действий программы
  • Импорт и экспорт данных

Основы работы с файлами

В Python для работы с файлами используется встроенная функция open(), которая открывает файл и возвращает объект файла. После завершения работы с файлом его необходимо закрыть с помощью метода close().

# Базовый синтаксис работы с файлами
# open(имя_файла, режим_доступа)

# Открытие файла для чтения
file = open("example.txt", "r")
# Чтение содержимого
content = file.read()
print(content)
# Закрытие файла
file.close()

Однако рекомендуется использовать конструкцию with, которая автоматически закрывает файл после выхода из блока, даже если произошла ошибка:

# Рекомендуемый способ работы с файлами
with open("example.txt", "r") as file:
    content = file.read()
    print(content)
# Файл автоматически закрывается после выхода из блока with

Режимы открытия файлов

Функция open() принимает различные режимы доступа к файлу:

Режим Описание
"r" Чтение (режим по умолчанию). Файл должен существовать.
"w" Запись. Создает новый файл или перезаписывает существующий.
"a" Добавление. Добавляет данные в конец файла, создает файл если он не существует.
"x" Эксклюзивное создание. Создает новый файл, выдает ошибку если файл уже существует.
"b" Бинарный режим (используется вместе с другими режимами, например "rb", "wb").
"t" Текстовый режим (режим по умолчанию).
"+" Чтение и запись (используется вместе с другими режимами, например "r+", "w+").

Примечание: Режимы можно комбинировать. Например, "rb" открывает файл для чтения в бинарном режиме, а "w+" открывает файл для чтения и записи, создавая новый файл или перезаписывая существующий.

Чтение из файлов

Существует несколько способов чтения данных из файла:

# Создадим тестовый файл для примеров
with open("planets.txt", "w") as file:
    file.write("Меркурий\nВенера\nЗемля\nМарс\nЮпитер\nСатурн\nУран\nНептун")

# Чтение всего файла целиком
with open("planets.txt", "r") as file:
    content = file.read()
    print("Содержимое файла:")
    print(content)

# Чтение файла построчно с помощью readlines()
with open("planets.txt", "r") as file:
    lines = file.readlines()
    print("\nСписок строк:")
    for i, line in enumerate(lines):
        print(f"{i+1}. {line.strip()}")

# Чтение файла построчно с помощью цикла
with open("planets.txt", "r") as file:
    print("\nПланеты Солнечной системы:")
    for line in file:
        print(f"- {line.strip()}")

# Чтение определенного количества символов
with open("planets.txt", "r") as file:
    first_10 = file.read(10)
    print("\nПервые 10 символов:")
    print(first_10)

Перемещение по файлу

Для перемещения по файлу используются методы seek() и tell():

with open("planets.txt", "r") as file:
    # Чтение первых 5 символов
    start = file.read(5)
    print(f"Первые 5 символов: '{start}'")
    
    # Получение текущей позиции
    position = file.tell()
    print(f"Текущая позиция: {position}")
    
    # Чтение следующих 5 символов
    next_5 = file.read(5)
    print(f"Следующие 5 символов: '{next_5}'")
    
    # Перемещение в начало файла
    file.seek(0)
    print(f"Новая позиция после seek(0): {file.tell()}")
    
    # Чтение первой строки после перемещения
    first_line = file.readline()
    print(f"Первая строка: '{first_line.strip()}'")
    
    # Перемещение на 20 байт от начала файла
    file.seek(20)
    pos_20 = file.read(10)
    print(f"10 символов с позиции 20: '{pos_20}'")

Запись в файлы

Для записи данных в файл используются методы write() и writelines():

# Запись строки в файл (создание нового файла или перезапись существующего)
with open("space_facts.txt", "w") as file:
    file.write("Факт 1: Солнце составляет 99.86% массы Солнечной системы.\n")
    file.write("Факт 2: На Венере день длиннее, чем год.\n")
    file.write("Факт 3: Юпитер имеет 79 известных спутников.\n")

# Чтение созданного файла
with open("space_facts.txt", "r") as file:
    print("Содержимое файла space_facts.txt:")
    print(file.read())

# Добавление данных в конец файла
with open("space_facts.txt", "a") as file:
    file.write("Факт 4: Один день на Меркурии длится примерно 176 земных дней.\n")
    file.write("Факт 5: На Марсе находится самая высокая гора в Солнечной системе - Олимп.\n")

# Запись списка строк с помощью writelines()
more_facts = [
    "Факт 6: Сатурн может плавать в воде из-за низкой плотности.\n",
    "Факт 7: Уран вращается "лёжа на боку".\n",
    "Факт 8: Нептун был открыт благодаря математическим расчётам.\n"
]

with open("space_facts.txt", "a") as file:
    file.writelines(more_facts)

# Проверка содержимого файла после всех операций
with open("space_facts.txt", "r") as file:
    print("\nОбновленное содержимое файла space_facts.txt:")
    for i, line in enumerate(file):
        print(line.strip())

Работа с путями к файлам

Для удобной работы с путями к файлам в Python используется модуль os.path или более современный модуль pathlib:

import os
from pathlib import Path

# Получение текущего рабочего каталога
current_dir = os.getcwd()
print(f"Текущий каталог: {current_dir}")

# Объединение путей с помощью os.path
data_file = os.path.join(current_dir, "data", "planets.csv")
print(f"Путь к файлу с данными: {data_file}")

# Проверка существования файла
print(f"Файл space_facts.txt существует: {os.path.exists('space_facts.txt')}")
print(f"Файл nonexistent.txt существует: {os.path.exists('nonexistent.txt')}")

# Получение информации о файле
if os.path.exists('space_facts.txt'):
    file_size = os.path.getsize('space_facts.txt')
    print(f"Размер файла space_facts.txt: {file_size} байт")
    
    base_name = os.path.basename('space_facts.txt')
    dir_name = os.path.dirname(os.path.abspath('space_facts.txt'))
    print(f"Имя файла: {base_name}")
    print(f"Каталог файла: {dir_name}")

# Использование pathlib (более современный подход)
print("\nРабота с pathlib:")
current_path = Path.cwd()
print(f"Текущий каталог: {current_path}")

file_path = current_path / "data" / "planets.csv"
print(f"Путь к файлу: {file_path}")

facts_file = Path('space_facts.txt')
print(f"Файл существует: {facts_file.exists()}")
if facts_file.exists():
    print(f"Размер файла: {facts_file.stat().st_size} байт")
    print(f"Имя файла: {facts_file.name}")
    print(f"Расширение: {facts_file.suffix}")
    print(f"Родительский каталог: {facts_file.parent.absolute()}")

Работа с CSV файлами

CSV (Comma-Separated Values) - популярный формат для хранения табличных данных. Python предоставляет модуль csv для работы с такими файлами:

import csv

# Создание CSV файла
planets_data = [
    ["Название", "Диаметр (км)", "Расстояние от Солнца (млн км)", "Количество спутников"],
    ["Меркурий", 4879, 57.9, 0],
    ["Венера", 12104, 108.2, 0],
    ["Земля", 12742, 149.6, 1],
    ["Марс", 6779, 227.9, 2],
    ["Юпитер", 139820, 778.5, 79],
    ["Сатурн", 116460, 1434.0, 82],
    ["Уран", 50724, 2871.0, 27],
    ["Нептун", 49244, 4495.0, 14]
]

# Запись данных в CSV файл
with open("planets.csv", "w", newline='') as file:
    writer = csv.writer(file)
    writer.writerows(planets_data)

print("CSV файл создан успешно.")

# Чтение данных из CSV файла
print("\nЧтение CSV файла:")
with open("planets.csv", "r") as file:
    reader = csv.reader(file)
    header = next(reader) # Чтение заголовка
    print(f"Заголовки: {', '.join(header)}")
    
    print("\nПланеты Солнечной системы:")
    for row in reader:
        print(f"{row[0]}: диаметр {row[1]} км, расстояние {row[2]} млн км, спутников: {row[3]}")

# Чтение CSV в список словарей
print("\nЧтение CSV в словари:")
with open("planets.csv", "r") as file:
    dict_reader = csv.DictReader(file)
    planets_list = list(dict_reader)

# Фильтрация и обработка данных
large_planets = [planet for planet in planets_list if int(planet["Диаметр (км)"]) > 50000]

print(f"Планеты с диаметром более 50000 км: {len(large_planets)}")
for planet in large_planets:
    print(f"- {planet['Название']} (диаметр: {planet['Диаметр (км)']} км)")

Работа с JSON файлами

JSON (JavaScript Object Notation) - популярный формат для хранения и обмена структурированными данными. Python предоставляет модуль json для работы с этим форматом:

import json

# Создание структуры данных для сохранения в JSON
solar_system = {
    "name": "Солнечная система",
    "star": {
        "name": "Солнце",
        "type": "G-type main-sequence",
        "diameter": 1392700,
        "age": 4.6e9
    },
    "planets": [
        {
            "name": "Меркурий",
            "type": "Каменистая планета",
            "diameter": 4879,
            "moons": []
        },
        {
            "name": "Венера",
            "type": "Каменистая планета",
            "diameter": 12104,
            "moons": []
        },
        {
            "name": "Земля",
            "type": "Каменистая планета",
            "diameter": 12742,
            "moons": ["Луна"]
        },
        {
            "name": "Марс",
            "type": "Каменистая планета",
            "diameter": 6779,
            "moons": ["Фобос", "Деймос"]
        }
    ]
}

# Запись данных в JSON файл
with open("solar_system.json", "w", encoding="utf-8") as file:
    json.dump(solar_system, file, ensure_ascii=False, indent=4)

# Чтение данных из JSON файла
with open("solar_system.json", "r", encoding="utf-8") as file:
    loaded_data = json.load(file)

for planet in loaded_data['planets']:
    moons_count = len(planet['moons'])
    moons_text = ", ".join(planet['moons']) if moons_count > 0 else "нет"

    "diameter": 139820,

# Сохранение обновленных данных
with open("solar_system_updated.json", "w", encoding="utf-8") as file:
    json.dump(loaded_data, file, ensure_ascii=False, indent=4)

print("\nОбновленный JSON файл создан успешно.")
print(f"Теперь в системе {len(loaded_data['planets'])} планет.")

Работа с бинарными файлами

Бинарные файлы содержат данные, которые не обязательно являются текстом. Для работы с ними используется режим "b":

# Запись бинарных данных
data = bytes([0, 1, 2, 3, 4, 5, 255])

with open("binary_data.bin", "wb") as file:
    file.write(data)

print("Бинарный файл создан.")

# Чтение бинарных данных
with open("binary_data.bin", "rb") as file:
    binary_data = file.read()

print(f"Прочитано {len(binary_data)} байт: {list(binary_data)}")

# Использование модуля struct для работы со структурированными бинарными данными
import struct

# Упаковка данных в бинарный формат
# 'i' - целое число (4 байта)
# 'f' - число с плавающей точкой (4 байта)
# '10s' - строка длиной 10 байт
packed_data = struct.pack('if10s', 42, 3.14, b'Hello Mars')

with open("struct_data.bin", "wb") as file:
    file.write(packed_data)

print("\nСтруктурированные данные записаны.")

# Чтение и распаковка структурированных данных
with open("struct_data.bin", "rb") as file:
    binary_data = file.read()
    unpacked_data = struct.unpack('if10s', binary_data)

print(f"Распакованные данные: {unpacked_data}")

def analyze_log(log_file):
    print(f"Анализ лог-файла: {log_file}")
    
    # Счетчики для разных типов сообщений
    info_count = 0
    warning_count = 0
    error_count = 0
    errors = []
    
    with open(log_file, "r") as file:
        for line in file:
            if " INFO " in line:
                info_count += 1
            elif " WARNING " in line:
                warning_count += 1
            elif " ERROR " in line:
                error_count += 1
                # Извлечение сообщения об ошибке
                error_message = line.split(" ERROR ")[1].strip()
                errors.append(error_message)

    # Вывод статистики
    print(f"\nСтатистика сообщений:")
    print(f"- INFO: {info_count}")
    print(f"- WARNING: {warning_count}")
    print(f"- ERROR: {error_count}")

    if errors:
        print("\nОбнаруженные ошибки:")
        for i, error in enumerate(errors, 1):
            print(f"{i}. {error}")

    # Создание отчета
    report_file = "log_report.txt"
    with open(report_file, "w") as file:
        file.write(f"Отчет по анализу лог-файла: {log_file}\n")
        file.write(f"\nСтатистика сообщений:\n")
        file.write(f"- INFO: {info_count}\n")
        file.write(f"- WARNING: {warning_count}\n")
        file.write(f"- ERROR: {error_count}\n")

        if errors:
            file.write("\nОбнаруженные ошибки:\n")
            for i, error in enumerate(errors, 1):
                file.write(f"{i}. {error}\n")

    print(f"\nОтчет сохранен в файл: {report_file}")

def analyze_planets(csv_file):
    print(f"Анализ данных о планетах из файла: {csv_file}")
    
    # Чтение данных из CSV файла
    planets = []
    with open(csv_file, "r") as file:
        reader = csv.DictReader(file)
        for row in reader:
            # Преобразование строковых значений в числовые
            row["Диаметр (км)"] = int(row["Диаметр (км)"])
            row["Расстояние от Солнца (млн км)"] = float(row["Расстояние от Солнца (млн км)"])
            row["Количество спутников"] = int(row["Количество спутников"])
            planets.append(row)

    # Анализ данных
    total_planets = len(planets)
    total_moons = sum(planet["Количество спутников"] for planet in planets)
    avg_diameter = sum(planet["Диаметр (км)"] for planet in planets) / total_planets

    # Нахождение самой большой и самой маленькой планеты
    largest_planet = max(planets, key=lambda p: p["Диаметр (км)"])
    smallest_planet = min(planets, key=lambda p: p["Диаметр (км)"])

    # Сортировка планет по расстоянию от Солнца
    sorted_by_distance = sorted(planets, key=lambda p: p["Расстояние от Солнца (млн км)"])

    # Вывод результатов анализа
    print(f"\nРезультаты анализа:")
    print(f"Количество планет: {total_planets}")
    print(f"Общее количество спутников: {total_moons}")
    print(f"Средний диаметр планет: {avg_diameter:.2f} км")
    print(f"Самая большая планета: {largest_planet['Название']} (диаметр: {largest_planet['Диаметр (км)']} км)")
    print(f"Самая маленькая планета: {smallest_planet['Название']} (диаметр: {smallest_planet['Диаметр (км)']} км)")

    print("\nПланеты в порядке удаления от Солнца:")
    for i, planet in enumerate(sorted_by_distance, 1):
        print(f"{i}. {planet['Название']} - {planet['Расстояние от Солнца (млн км)']} млн км")

    # Создание нового CSV файла с дополнительными данными
    output_file = "planets_analysis.csv"
    with open(output_file, "w", newline='') as file:
        fieldnames = ["Название", "Диаметр (км)", "Расстояние от Солнца (млн км)", "Количество спутников", "Тип", "Относительный размер"]
        writer = csv.DictWriter(file, fieldnames=fieldnames)
        writer.writeheader()

        for planet in planets:
            # Определение типа планеты
            if planet["Диаметр (км)"] > 50000:
                planet_type = "Газовый гигант"
            else:
                planet_type = "Каменистая планета"

            # Определение относительного размера
            earth_diameter = 12742 # диаметр Земли в км
            relative_size = planet["Диаметр (км)"] / earth_diameter

            # Добавление новых данных
            planet["Тип"] = planet_type
            planet["Относительный размер"] = f"{relative_size:.2f} x Земля"

            writer.writerow(planet)

    print(f"\nРасширенные данные сохранены в файл: {output_file}")

# Запуск анализа данных о планетах
