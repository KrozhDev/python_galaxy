Что такое списки?

Список (list) в Python - это упорядоченная коллекция элементов, которые могут быть разных типов. Списки являются изменяемыми (mutable), то есть их содержимое можно изменять после создания.

Аналогия: Представьте список как контейнер с пронумерованными ячейками. В каждую ячейку можно положить любой объект - число, текст, другой список и т.д. Вы можете добавлять новые ячейки, удалять существующие или изменять их содержимое.

Создание списков

Списки в Python создаются с помощью квадратных скобок [] или функции list():

# Пустой список
empty_list = []
also_empty = list()

# Список с элементами
planets = ["Меркурий", "Венера", "Земля", "Марс"]

# Список с элементами разных типов
mixed_data = ["Земля", 1, 9.8, True]

# Вложенные списки
nested_list = [[1, 2], [3, 4], [5, 6]]

# Создание списка с помощью функции list()
chars = list("Привет") # ['П', 'р', 'и', 'в', 'е', 'т']

# Вывод списков
print("Планеты:", planets)
print("Смешанные данные:", mixed_data)
print("Вложенный список:", nested_list)
print("Список символов:", chars)
Терминал
Планеты: ['Меркурий', 'Венера', 'Земля', 'Марс']
Смешанные данные: ['Земля', 1, 9.8, True]
Вложенный список: [[1, 2], [3, 4], [5, 6]]
Список символов: ['П', 'р', 'и', 'в', 'е', 'т']

Доступ к элементам списка

Элементы списка индексируются, начиная с 0. Для доступа к элементу используются квадратные скобки с указанием индекса:

0 -4 Меркурий
1 -3 Венера
2 -2 Земля
3 -1 Марс
planets = ["Меркурий", "Венера", "Земля", "Марс"]

# Доступ по положительному индексу (от начала списка)
print("Первая планета:", planets[0])
print("Третья планета:", planets[2])

# Доступ по отрицательному индексу (от конца списка)
print("Последняя планета:", planets[-1])
print("Предпоследняя планета:", planets[-2])
Терминал
Первая планета: Меркурий
Третья планета: Земля
Последняя планета: Марс
Предпоследняя планета: Земля

Внимание: Если индекс выходит за пределы списка, Python выдаст ошибку IndexError. Например, если в списке 4 элемента, попытка доступа к planets[4] или planets[-5] вызовет ошибку.

Срезы списков

Срезы (slices) позволяют извлекать подсписки из списка:

planets = ["Меркурий", "Венера", "Земля", "Марс", "Юпитер", "Сатурн"]

# Синтаксис среза: список[начало:конец:шаг]
# Конец не включается в результат!

# Первые три планеты
print("Первые три планеты:", planets[0:3])

# Со второй по четвертую
print("Со второй по четвертую:", planets[1:4])

# От начала до третьей (не включая)
print("От начала до третьей:", planets[:3])

# От третьей до конца
print("От третьей до конца:", planets[2:])

# Копия всего списка
print("Копия списка:", planets[:])

# Каждая вторая планета
print("Каждая вторая планета:", planets[:6:2])

# Список в обратном порядке
print("Список в обратном порядке:", planets[::-1])

Изменение элементов списка

Списки в Python являются изменяемыми, поэтому мы можем изменять их элементы:

planets = ["Меркурий", "Венера", "Земля", "Марс"]
print("Исходный список:", planets)

# Изменение одного элемента
planets[0] = "МЕРКУРИЙ"
print("После изменения первого элемента:", planets)

# Изменение нескольких элементов с помощью среза
planets[1:3] = ["ВЕНЕРА", "ЗЕМЛЯ"]
print("После изменения среза:", planets)
Терминал
Исходный список: ['Меркурий', 'Венера', 'Земля', 'Марс']
После изменения первого элемента: ['МЕРКУРИЙ', 'Венера', 'Земля', 'Марс']
После изменения среза: ['МЕРКУРИЙ', 'ВЕНЕРА', 'ЗЕМЛЯ', 'Марс']

Основные методы списков

Python предоставляет множество встроенных методов для работы со списками:

Добавление элементов

planets = ["Меркурий", "Венера", "Земля"]
print("Исходный список:", planets)

# Добавление элемента в конец списка
planets.append("Марс")
print("После append():", planets)

# Вставка элемента по индексу
planets.insert(1, "Меркурий-B")
print("После insert():", planets)

# Расширение списка другим списком
outer_planets = ["Юпитер", "Сатурн"]
planets.extend(outer_planets)
print("После extend():", planets)
Терминал
Исходный список: ['Меркурий', 'Венера', 'Земля']
После append(): ['Меркурий', 'Венера', 'Земля', 'Марс']
После insert(): ['Меркурий', 'Меркурий-B', 'Венера', 'Земля', 'Марс']
После extend(): ['Меркурий', 'Меркурий-B', 'Венера', 'Земля', 'Марс', 'Юпитер', 'Сатурн']

Разница между append() и extend(): Метод append() добавляет один элемент в конец списка, а extend() добавляет все элементы из другой последовательности. Если вы используете append() для добавления списка, то список будет добавлен как один элемент (список внутри списка).

Удаление элементов

planets = ["Меркурий", "Венера", "Земля", "Марс", "Юпитер", "Сатурн"]
print("Исходный список:", planets)

# Удаление элемента по индексу
del planets[1]
print("После del planets[1]:", planets)

# Удаление элемента по значению
planets.remove("Марс")
print("После remove('Марс'):", planets)

# Удаление и возврат элемента по индексу
last_planet = planets.pop()
print(f"Удаленный элемент: {last_planet}")
print("После pop():", planets)

# Удаление и возврат элемента по указанному индексу
first_planet = planets.pop(0)
print(f"Удаленный элемент с индексом 0: {first_planet}")
print("После pop(0):", planets)

# Очистка списка
planets.clear()
print("После clear():", planets)
Терминал
Исходный список: ['Меркурий', 'Венера', 'Земля', 'Марс', 'Юпитер', 'Сатурн']
После del planets[1]: ['Меркурий', 'Земля', 'Марс', 'Юпитер', 'Сатурн']
После remove('Марс'): ['Меркурий', 'Земля', 'Юпитер', 'Сатурн']
Удаленный элемент: Сатурн
После pop(): ['Меркурий', 'Земля', 'Юпитер']
Удаленный элемент с индексом 0: Меркурий
После pop(0): ['Земля', 'Юпитер']
После clear(): []

Внимание: Метод remove() удаляет только первое вхождение указанного значения. Если в списке несколько одинаковых элементов, остальные останутся на месте.

Поиск и подсчет элементов

planets = ["Меркурий", "Венера", "Земля", "Марс", "Юпитер", "Венера"]

count = planets.count("Венера")

Сортировка и изменение порядка элементов

planets = ["Меркурий", "Венера", "Земля", "Марс", "Юпитер"]
print("Исходный список:", planets)

# Сортировка списка по алфавиту
planets.sort()
print("После sort():", planets)

# Сортировка в обратном порядке
planets.sort(reverse=True)
print("После sort(reverse=True):", planets)

# Изменение порядка элементов на обратный
planets.reverse()
print("После reverse():", planets)
Терминал
Исходный список: ['Меркурий', 'Венера', 'Земля', 'Марс', 'Юпитер']
После sort(): ['Венера', 'Земля', 'Марс', 'Меркурий', 'Юпитер']
После sort(reverse=True): ['Юпитер', 'Меркурий', 'Марс', 'Земля', 'Венера']
После reverse(): ['Венера', 'Земля', 'Марс', 'Меркурий', 'Юпитер']

Важно: Методы sort() и reverse() изменяют исходный список. Если вы хотите получить новый отсортированный список, не изменяя исходный, используйте функцию sorted().

Копирование списков

original = ["Меркурий", "Венера", "Земля"]

# Неправильный способ копирования (создает ссылку на тот же список)
wrong_copy = original
wrong_copy.append("Марс")
print("Оригинал после изменения wrong_copy:", original)

# Правильные способы копирования
# 1. Метод copy()
copy1 = original.copy()

# 2. Функция list()
copy2 = list(original)

# 3. Срез [:]
copy3 = original[:]

# Проверка, что копии независимы
copy1.append("Юпитер")
print("Оригинал после изменения copy1:", original)
print("copy1:", copy1)
Терминал
Оригинал после изменения wrong_copy: ['Меркурий', 'Венера', 'Земля', 'Марс']
Оригинал после изменения copy1: ['Меркурий', 'Венера', 'Земля', 'Марс']
copy1: ['Меркурий', 'Венера', 'Земля', 'Марс', 'Юпитер']

Внимание: Все три метода копирования (copy(), list(), срез [:]) создают поверхностную копию. Это означает, что если список содержит вложенные списки или другие изменяемые объекты, они будут ссылаться на те же объекты. Для создания полной копии (включая вложенные объекты) используйте модуль copy и функцию deepcopy().

Списочные включения (List Comprehensions)

Списочные включения - это компактный способ создания списков на основе существующих последовательностей:

# Создание списка квадратов чисел от 1 до 10

# Обычный способ с циклом for
squares = []
for i in range(1, 11):
    squares.append(i ** 2)
print("Квадраты (обычный способ):", squares)

# Тот же результат с использованием списочного включения
squares_comp = [i ** 2 for i in range(1, 11)]
print("Квадраты (списочное включение):", squares_comp)

# Списочное включение с условием
# Только четные квадраты
even_squares = [i ** 2 for i in range(1, 11) if i % 2 == 0]
print("Четные квадраты:", even_squares)
Терминал
Квадраты (обычный способ): [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
Квадраты (списочное включение): [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
Четные квадраты: [4, 16, 36, 64, 100]

Списочные включения могут быть очень полезны для преобразования данных:

# Преобразование строк в верхний регистр
planets = ["меркурий", "венера", "земля", "марс"]
upper_planets = [planet.upper() for planet in planets]
print("Планеты в верхнем регистре:", upper_planets)

# Извлечение длин строк
planet_lengths = [len(planet) for planet in planets]
print("Длины названий планет:", planet_lengths)

# Создание списка кортежей
planet_info = [(planet, len(planet)) for planet in planets]
print("Информация о планетах:", planet_info)

Пример 3: Простая игра с использованием списков

import random

def space_quiz():
    # Список вопросов и ответов
    questions = [
        {
            "question": "Какая планета ближайшая к Солнцу?",
            "options": ["Венера", "Меркурий", "Марс", "Земля"],
            "answer": 1 # Индекс правильного ответа
        },
        {
            "question": "Какая планета известна своими кольцами?",
            "options": ["Юпитер", "Уран", "Нептун", "Сатурн"],
            "answer": 3
        },
        {
            "question": "Какая планета называется 'красной планетой'?",
            "options": ["Венера", "Марс", "Юпитер", "Меркурий"],
            "answer": 1
        }
    ]
    
    # Перемешиваем вопросы
    random.shuffle(questions)
    
    score = 0
    
    print("=== Космическая викторина ===")
    print("Ответьте на следующие вопросы о планетах Солнечной системы:\n")
    
    for i, q in enumerate(questions):
        print(f"Вопрос {i+1}: {q['question']}")
        
        for j, option in enumerate(q["options"]):
            print(f" {j+1}. {option}")
        
        try:
            user_answer = int(input("\nВаш ответ (введите номер): ")) - 1
            
            if user_answer == q["answer"]:
                print("Правильно! +1 балл\n")
                score += 1
            else:
                print(f"Неправильно. Правильный ответ: {q['options'][q['answer']]}\n")
        except (ValueError, IndexError):
            print("Неверный ввод. Пропускаем вопрос.\n")
    
    print(f"Викторина завершена! Ваш результат: {score}/{len(questions)}")
    
    if score == len(questions):
        print("Отлично! Вы настоящий знаток космоса!")
    elif score >= len(questions) / 2:
        print("Хороший результат! Вы много знаете о планетах.")
    else     else:
        print("Стоит подучить астрономию! Удачи в следующий раз.")

# Запуск викторины
