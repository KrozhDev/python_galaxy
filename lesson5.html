Что такое условные операторы?

Условные операторы позволяют программе выполнять разные действия в зависимости от того, выполняется ли определенное условие. Это основа принятия решений в программировании.

Аналогия: Представьте, что вы стоите на развилке дорог. Если идет дождь, вы выбираете дорогу с укрытием. Если солнечно, вы идете по живописному маршруту. Условные операторы работают так же - они направляют программу по разным путям в зависимости от условий.

Оператор if

Самый простой условный оператор - if. Он выполняет блок кода только если условие истинно:

# Простой пример использования if
temperature = -5

if temperature < 0:
    print("Внимание! Температура ниже нуля.")

print("Программа завершена.")
Терминал
Внимание! Температура ниже нуля.
Программа завершена.
temperature < 0?
Да
Вывести предупреждение
Нет
Вывести "Программа завершена"

Важно: В Python отступы имеют значение! Код внутри блока if должен быть с отступом (обычно 4 пробела). Это показывает, какой код выполняется при истинном условии.

Операторы if-else

Если нужно выполнить одно действие при истинном условии и другое при ложном, используется конструкция if-else:

fuel_level = 20

if fuel_level > 30:
    print("Топлива достаточно для путешествия.")
else:
    print("Внимание! Низкий уровень топлива.")

Операторы if-elif-else

Когда нужно проверить несколько условий, используется конструкция if-elif-else (elif - сокращение от "else if"):

radiation_level = 75

if radiation_level < 50:
    print("Уровень радиации безопасен.")
elif radiation_level < 100:
    print("Внимание! Повышенный уровень радиации.")
else:
    print("ОПАСНОСТЬ! Критический уровень радиации!")

Вы можете использовать сколько угодно блоков elif для проверки различных условий. Python проверяет условия по порядку и выполняет код первого истинного условия.

Операторы сравнения

Для создания условий используются операторы сравнения:

Оператор Описание Пример
== Равно x == y
!= Не равно x != y
> Больше x > y
< Меньше x < y
>= Больше или равно x >= y
<= Меньше или равно x <= y

Внимание: Не путайте оператор присваивания = с оператором сравнения ==. Это одна из самых распространенных ошибок в программировании!

Логические операторы

Для создания сложных условий используются логические операторы:

Оператор Описание Пример
and Логическое И (оба условия должны быть истинными) x > 0 and x < 10
or Логическое ИЛИ (хотя бы одно условие должно быть истинным) x < 0 or x > 10
not Логическое НЕ (инвертирует условие) not x > 10
temperature = 25
pressure = 980

# Проверка нескольких условий с помощью and
if temperature > 20 and pressure > 1000:
    print("Условия оптимальные.")
elif temperature > 20 or pressure > 1000:
    print("Условия приемлемые.")
else:
    print("Условия неблагоприятные.")
Терминал
Условия приемлемые.

Вложенные условия

Условные операторы можно вкладывать друг в друга для создания более сложной логики:

oxygen_level = 85
has_spacesuit = True

if oxygen_level < 90:
    print("Внимание! Пониженный уровень кислорода.")
    
    if has_spacesuit:
        print("Наденьте скафандр для безопасности.")
    else:
        print("ОПАСНОСТЬ! Немедленно эвакуируйтесь!")
else:
    print("Уровень кислорода в норме.")
Терминал
Внимание! Пониженный уровень кислорода.
Наденьте скафандр для безопасности.

Тернарный оператор

Python также поддерживает тернарный оператор - краткую форму записи условия в одну строку:

temperature = -10

# Обычная запись с if-else
if temperature > 0:
    weather = "тепло"
else:
    weather = "холодно"

# Тернарный оператор (то же самое, но короче)
weather = "тепло" if temperature > 0 else "холодно"

print(f"Сегодня {weather}.")
Терминал
Сегодня холодно.

Синтаксис тернарного оператора: значение_если_истина if условие else значение_если_ложь

Проверка наличия элемента с помощью in

Оператор in позволяет проверить, содержится ли элемент в последовательности (список, строка, кортеж и т.д.):

allowed_planets = ["Земля", "Марс", "Луна", "Венера"]
destination = "Юпитер"

if destination in allowed_planets:
    print(f"Полет на {destination} разрешен.")
else:
    print(f"Полет на {destination} запрещен. Требуется специальное разрешение.")
Терминал
Полет на Юпитер запрещен. Требуется специальное разрешение.

Пример 1: Проверка доступа

# Система проверки доступа
username = input("Введите имя пользователя: ")

# Правильные учетные данные
correct_username = "admin"
correct_password = "12345"

if username == correct_username and password == correct_password:
    print("Доступ разрешен. Добро пожаловать в систему!")
elif username == correct_username:
    print("Ошибка: неверный пароль.")
else:
    print("Ошибка: пользователь не найден.")

Пример 2: Классификация планет

# Классификация планет по размеру
diameter = float(input("Введите диаметр планеты (км): "))

if diameter < 5000:
    planet_type = "карликовая планета"
elif diameter < 10000:
    planet_type = "планета земной группы"
elif diameter < 50000:
    planet_type = "газовый гигант"
else:
    planet_type = "сверхгигант"

print(f"Объект с диаметром {diameter} км классифицируется как {planet_type}.")

Пример 3: Калькулятор стоимости доставки

print("Калькулятор стоимости межпланетной доставки")
weight = float(input("Введите вес груза (кг): "))
destination = input("Введите пункт назначения (Луна/Марс/Венера): ")

# Базовая стоимость доставки
base_cost = 1000

# Коэффициенты для разных планет
if destination.lower() == "луна":
    coefficient = 1.0
elif destination.lower() == "марс":
    coefficient = 2.5
elif destination.lower() == "венера":
    coefficient = 3.0
else:
    print("Ошибка: неизвестный пункт назначения")
    coefficient = 0

# Надбавка за вес
if weight < 10:
    weight_surcharge = 100
elif weight < 100:
    weight_surcharge = weight * 25
else:
    weight_surcharge = weight * 50

# Расчет итоговой стоимости
total_cost = (base_cost + weight_surcharge) * coefficient

if coefficient > 0:
    print(f"Стоимость доставки {weight} кг на {destination}: ${total_cost:.2f}")
Терминал
Калькулятор стоимости межпланетной доставки
Введите вес груза (кг): 50
Введите пункт назначения (Луна/Марс/Венера): Марс
Стоимость доставки 50.0 кг на Марс: \$3750.00

Для получения кода доступа к следующей планете, создай программу "Диагностика космического корабля", которая:

  1. Запрашивает у пользователя уровень топлива (0-100%)
  2. Запрашивает температуру двигателя (в градусах Цельсия)
  3. Запрашивает давление в кабине (в кПа, нормальное значение около 100 кПа)
  4. Анализирует состояние корабля и выводит диагностическое сообщение

Правила диагностики:

  • Если топлива меньше 20% - "Критический уровень топлива"
  • Если температура двигателя выше 110°C - "Перегрев двигателя"
  • Если давление в кабине ниже 80 кПа или выше 120 кПа - "Разгерметизация кабины"
  • Если все показатели в норме - "Все системы работают нормально"
  • Если есть несколько проблем, нужно вывести все соответствующие предупреждения
  • В конце нужно вывести общий статус: "ПОЛЕТ ВОЗМОЖЕН" (если все в норме или только низкий уровень топлива) или "ПОЛЕТ НЕВОЗМОЖЕН" (при любых других проблемах)
