Здесь мы изучим словари (dictionaries) - одну из самых мощных и гибких структур данных в Python. Словари позволяют хранить данные в виде пар "ключ-значение", что делает их идеальными для работы с именованными данными и сложными структурами.

Наша задача - освоить создание и манипуляцию словарями, чтобы эффективно управлять данными в нашей миссии по восстановлению Галактической Сети.

Что такое словари?

Словарь (dictionary) в Python - это неупорядоченная коллекция пар "ключ-значение". Каждый ключ связан с определенным значением, что позволяет быстро находить, добавлять и изменять данные. Словари являются изменяемыми (mutable) и могут содержать элементы разных типов.

Аналогия: Представьте словарь как настоящий словарь или телефонную книгу. Чтобы найти номер телефона (значение), вы ищете имя человека (ключ). Ключи в словаре должны быть уникальными, как и имена в телефонной книге, а значения могут повторяться.

Создание словарей

Словари в Python создаются с помощью фигурных скобок {} или функции dict():

# Пустой словарь
empty_dict = {}
also_empty = dict()

# Словарь с элементами
planet = {
    "name": "Земля",
    "diameter": 12742,
    "distance_from_sun": 149.6,
    "has_moon": True
}

# Создание словаря с помощью функции dict()
planet2 = dict(
    name="Марс",
    diameter=6779,
    distance_from_sun=227.9,
    has_moon=True
)

# Создание словаря из списка кортежей
planet_items = [("name", "Венера"), ("diameter", 12104), ("has_moon", False)]
planet3 = dict(planet_items)

# Вывод словарей
print("Земля:", planet)
print("Марс:", planet2)
print("Венера:", planet3)
Терминал
Земля: {'name': 'Земля', 'diameter': 12742, 'distance_from_sun': 149.6, 'has_moon': True}
Марс: {'name': 'Марс', 'diameter': 6779, 'distance_from_sun': 227.9, 'has_moon': True}
Венера: {'name': 'Венера', 'diameter': 12104, 'has_moon': False}

Важно: Ключи словаря должны быть неизменяемыми (immutable) типами данных, такими как строки, числа или кортежи. Списки и другие словари не могут быть ключами, так как они изменяемы. Значения могут быть любого типа, включая списки, словари и другие сложные объекты.

Доступ к элементам словаря

Для доступа к значениям словаря используются квадратные скобки с указанием ключа или метод get():

planet = {
    "name": "Земля",
    "diameter": 12742,
    "distance_from_sun": 149.6,
    "has_moon": True
}

# Доступ с помощью квадратных скобок
print("Название планеты:", planet["name"])
print("Диаметр планеты:", planet["diameter"], "км")

# Доступ с помощью метода get()
# get() возвращает None, если ключ не найден
print("Расстояние от Солнца:", planet.get("distance_from_sun"), "млн км")

# get() с значением по умолчанию
print("Атмосфера:", planet.get("atmosphere", "Нет данных"))
Терминал
Название планеты: Земля
Диаметр планеты: 12742 км
Расстояние от Солнца: 149.6 млн км
Атмосфера: Нет данных

Внимание: Если ключ не существует, доступ через квадратные скобки вызовет ошибку KeyError. Метод get() безопаснее, так как он возвращает None или указанное значение по умолчанию, если ключ не найден.

Изменение элементов словаря

Словари в Python являются изменяемыми, поэтому мы можем добавлять, изменять и удалять их элементы:

planet = {
    "name": "Марс",
    "diameter": 6779
}
print("Исходный словарь:", planet)

# Добавление нового элемента
planet["moons"] = ["Фобос", "Деймос"]
print("После добавления спутников:", planet)

# Изменение существующего элемента
planet["diameter"] = 6780 # Уточненное значение
print("После изменения диаметра:", planet)

# Добавление нескольких элементов с помощью update()
planet.update({
    "distance_from_sun": 227.9,
    "has_rings": False
})
print("После update():", planet)

Удаление элементов словаря

planet = {
    "name": "Юпитер",
    "diameter": 139820,
    "distance_from_sun": 778.5,
    "has_rings": True,
    "moons_count": 79
}
print("Исходный словарь:", planet)

# Удаление элемента с помощью del
del planet["has_rings"]
print("После del:", planet)

# Удаление и возврат элемента с помощью pop()
moons = planet.pop("moons_count")
print(f"Удаленное значение: {moons}")
print("После pop():", planet)

# Удаление и возврат произвольной пары с помощью popitem()
# В Python 3.7+ popitem() удаляет последнюю добавленную пару
item = planet.popitem()
print(f"Удаленная пара: {item}")
print("После popitem():", planet)

# Очистка словаря
planet.clear()
print("После clear():", planet)

Проверка наличия ключа

Для проверки наличия ключа в словаре используется оператор in:

planet = {
    "name": "Сатурн",
    "diameter": 116460,
    "has_rings": True
}

# Проверка наличия ключа
print("Есть ли ключ 'name'?", "name" in planet)
print("Есть ли ключ 'moons'?", "moons" in planet)

# Проверка отсутствия ключа
print("Отсутствует ли ключ 'temperature'?", "temperature" not in planet)

# Безопасный доступ к значению с проверкой
if "moons" in planet:
    print("Спутники:", planet["moons"])
else:
    print("Информация о спутниках отсутствует")

Итерация по словарю

Существует несколько способов перебора элементов словаря:

planet = {
    "name": "Нептун",
    "diameter": 49244,
    "distance_from_sun": 4495.1,
    "has_rings": True,
    "moons": ["Тритон", "Нереида", "Наяда"]
}

# Итерация по ключам (поведение по умолчанию)
print("Ключи словаря:")
for key in planet:
    print(f"- {key}")

# Итерация по ключам с помощью keys()
print("\nКлючи с помощью keys():")
for key in planet.keys():
    print(f"- {key}")

# Итерация по значениям с помощью values()
print("\nЗначения словаря:")
for value in planet.values():
    print(f"- {value}")

# Итерация по парам ключ-значение с помощью items()
print("\nПары ключ-значение:")
for key, value in planet.items():
    print(f"- {key}: {value}")

Вложенные словари

Словари могут содержать другие словари в качестве значений, что позволяет создавать сложные структуры данных:

# Словарь с информацией о планетах
solar_system = {
    "Mercury": {
        "diameter": 4879,
        "distance_from_sun": 57.9,
        "moons": []
    },
    "Venus": {
        "diameter": 12104,
        "distance_from_sun": 108.2,
        "moons": []
    },
    "Earth": {
        "diameter": 12742,
        "distance_from_sun": 149.6,
        "moons": ["Moon"]
    }
}

# Доступ к вложенным данным
print(f"Диаметр Земли: {solar_system['Earth']['diameter']} км")
print(f"Расстояние от Венеры до Солнца: {solar_system['Venus']['distance_from_sun']} млн км")
print(f"Спутники Земли: {', '.join(solar_system['Earth']['moons'])}")

# Добавление новой планеты
solar_system["Mars"] = {
    "diameter": 6779,
    "distance_from_sun": 227.9,
    "moons": ["Phobos", "Deimos"]
}

# Перебор вложенных словарей
print("\nИнформация о планетах:")
for planet_name, planet_info in solar_system.items():
    print(f"\n{planet_name}:")
    print(f" Диаметр: {planet_info['diameter']} км")
    print(f" Расстояние от Солнца: {planet_info['distance_from_sun']} млн км")
    if planet_info['moons']:
        print(f" Спутники: {', '.join(planet_info['moons'])}")
    else:
        print(" Спутники: нет")

Словарные включения (Dictionary Comprehensions)

Словарные включения - это компактный способ создания словарей, аналогичный списочным включениям:

# Создание словаря с помощью словарного включения

# Словарь квадратов чисел
squares = {x: x**2 for x in range(1, 6)}
print("Словарь квадратов:", squares)

# Словарь с условием
even_squares = {x: x**2 for x in range(1, 11) if x % 2 == 0}
print("Словарь четных квадратов:", even_squares)

# Преобразование списка в словарь
planets = ["Mercury", "Venus", "Earth", "Mars"]
planet_lengths = {planet: len(planet) for planet in planets}
print("Длины названий планет:", planet_lengths)

# Обмен ключей и значений
original = {"a": 1, "b": 2, "c": 3}
inverted = {value: key for key, value in original.items()}
print("Исходный словарь:", original)
print("Инвертированный словарь:", inverted)

values() Возвращает представление значений for v in planet.values()

Пример 1: Подсчет частоты слов

def count_words(text):
    # Разбиваем текст на слова и удаляем знаки препинания
    words = text.lower().replace(',', '').replace('.', '').replace('!', '').replace('?', '').split()
    
    # Создаем словарь для подсчета частоты
    word_count = {}
    
    # Подсчитываем частоту каждого слова
    for word in words:
        if word in word_count:
            word_count[word] += 1
        else:
            word_count[word] = 1
    
    # Альтернативный способ с использованием setdefault
    # for word in words:
    # word_count.setdefault(word, 0)
    # word_count[word] += 1
    
    return word_count

# Пример использования
text = """Космос - это огромное пространство за пределами Земли. Космос содержит планеты, звезды, галактики и многое другое. Люди изучают космос с помощью телескопов и космических аппаратов."""

word_frequency = count_words(text)

# Вывод результатов
print("Частота слов в тексте:")
for word, count in sorted(word_frequency.items(), key=lambda x: x[1], reverse=True):
    if len(word) > 3: # Пропускаем короткие слова
        print(f"{word}: {count}")

def display_catalog(catalog):
    print("\n===== КАТАЛОГ КОСМИЧЕСКИХ ОБЪЕКТОВ =====")
    for obj_id, obj_info in sorted(catalog.items()):
        print(f"\nID: {obj_id}")
        print(f"Название: {obj_info['name']}")
        print(f"Тип: {obj_info['type']}")
        if 'distance' in obj_info:
            print(f"Расстояние: {obj_info['distance']} световых лет")
        if 'description' in obj_info:
            print(f"Описание: {obj_info['description']}")

def add_object(catalog):
    print("\n===== ДОБАВЛЕНИЕ НОВОГО ОБЪЕКТА =====")
    obj_id = input("Введите ID объекта: ")
    
    if obj_id in catalog:
        print(f"Объект с ID {obj_id} уже существует!")
        return
    
    name = input("Введите название: ")
    obj_type = input("Введите тип (планета, звезда, галактика и т.д.): ")
    distance = input("Введите расстояние в световых годах (оставьте пустым, если неизвестно): ")
    description = input("Введите описание (оставьте пустым, если нет): ")
    
    # Создаем словарь с информацией об объекте
    obj_info = {
        "name": name,
        "type": obj_type
    }
    
    if distance:
        obj_info["distance"] = float(distance)
    
    if description:
        obj_info["description"] = description
    
    # Добавляем объект в каталог
    catalog[obj_id] = obj_info
    print(f"Объект {name} успешно добавлен в каталог!")

def search_objects(catalog):
    print("\n===== ПОИСК ОБЪЕКТОВ =====")
    search_term = input("Введите поисковый запрос: ").lower()
    
    found = False
    print("\nРезультаты поиска:")
    
    for obj_id, obj_info in catalog.items():
        if (search_term in obj_id.lower() or
            search_term in obj_info['name'].lower() or
            search_term in obj_info['type'].lower() or
            ('description' in obj_info and search_term in obj_info['description'].lower())):
            print(f"\nID: {obj_id}")
            print(f"Название: {obj_info['name']}")
            print(f"Тип: {obj_info['type']}")
            found = True
    
    if not found:
        print("Ничего не найдено.")
